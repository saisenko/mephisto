# Mephisto 🦉☕️ 

### An Interactive IDE for Learning CISC, RISC, and VLIW Architectures

**Starting with RISC... because legends begin somewhere!**

---

(Here will be Mephisto's logo, stay tuned :>)

**Mephisto** is an innovative, user-friendly IDE designed to unlock the mysteries of modern processor architectures. Created with a learning-first approach, Mephisto helps enthusiasts, students, and engineers dive deep into **RISC**, **CISC**, and **VLIW** architectures through hands-on exploration and interactive tools.

## Why Mephisto?

Processor architectures drive the tech world, but mastering them can feel like taming a storm! Mephisto is here to make that journey intuitive, engaging, and fun by starting with RISC and expanding to CISC and VLIW. With a sleek interface, real-time feedback, and detailed instruction breakdowns, Mephisto aims to be your go-to learning platform for low-level architecture.

---

## Features 🌟

- **RISC Instruction Simulator**  
  Begin with a fully interactive RISC instruction set, exploring and experimenting with each instruction type.

- **Future-Proof Design**  
  Mephisto is designed to easily incorporate **CISC** and **VLIW** architectures as the project evolves.

- **Live Feedback and Analysis**  
  Watch the behavior of each instruction in real time, complete with cycle counts, memory access visuals, and register effects.

- **Visual Debugger and Execution Flow**  
  Navigate through instructions step-by-step with our visual debugger, helping you trace code execution effortlessly.

- **Modular Design**  
  The modular backend and GUI make it easy for contributors to add new architectures or expand on existing features.

## Getting Started 🚀

1. **Clone the Repository**  
    ```bash
    git clone https://github.com/yourusername/Mephisto.git
    cd Mephisto
    ```
2. **Build the Project**
    ```bash
    mkdir build && cd build
    cmake ..
    make
    ```
3. **Run Mephisto**
    ```bash
    python3 gui/main.py
    ```
